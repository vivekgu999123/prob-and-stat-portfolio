Experiment 1: Data Visualization

Objective: To visualize data using histogram and boxplot.


import pandas as pd
import matplotlib.pyplot as plt

data = pd.read_csv("Lungdata.csv")

plt.hist(data['Age'], bins=10)
plt.xlabel("Age")
plt.ylabel("Frequency")
plt.title("Histogram of Age")
plt.show()

plt.boxplot(data['LungCap'])
plt.title("Boxplot of Lung Capacity")
plt.show()

Experiment 2: Measures of Central Tendency and Dispersion

Objective: To compute mean, variance and standard deviation.


import numpy as np

data = [12, 15, 18, 20, 22, 25]

mean = np.mean(data)
variance = np.var(data)
std_dev = np.std(data)

print("Mean:", mean)
print("Variance:", variance)
print("Standard Deviation:", std_dev)

Experiment 3: Binomial Distribution

Objective: To compute probability using Binomial distribution.


from scipy.stats import binom

n = 10
p = 0.5

probability = binom.pmf(4, n, p)
print("Probability:", probability)

Experiment 4: Poisson Distribution

Objective: To compute probability using Poisson distribution.


from scipy.stats import poisson

lam = 3

probability = poisson.pmf(2, lam)
print("Probability:", probability)

Experiment 5: Normal Distribution

Objective: To find probability using Normal distribution.


from scipy.stats import norm

mean = 50
std = 5

probability = norm.cdf(55, mean, std)
print("Probability:", probability)

Experiment 6: Z-Test for Mean

Objective: To test hypothesis about population mean.


from scipy.stats import norm
import math

x_bar = 52
mu = 50
sigma = 4
n = 36

z = (x_bar - mu) / (sigma / math.sqrt(n))
p_value = 1 - norm.cdf(z)

print("Z-value:", z)
print("P-value:", p_value)

Experiment 7: t-Test for Difference of Means

Objective: To test difference between two sample means.


from scipy.stats import ttest_ind

group1 = [12, 14, 15, 16, 18]
group2 = [10, 11, 13, 14, 15]

t_stat, p_val = ttest_ind(group1, group2)

print("t-statistic:", t_stat)
print("p-value:", p_val)
